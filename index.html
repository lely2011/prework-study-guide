HTML

  • HTML stands for Hypertext Markup Language
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user.

CSS

  • CCS stands for Cascading Style Sheets
  • The head element contains language used to describe the presentation
  • The body element represents how each element on webpage is rendered

Git

  • Git Status: Checks what branch we are currently on
  • Git Checkout -b branch-name: creates a new branch and switches to it
  • Git is an open source distributed verson control system
  • Git allows web developers to collaborate on projects withouth overwriting each others work
  • Git also allows developers to revert to previous versions of the code if problem is discovered

JavaScript

  • JavaScript creates interactive effects within web browers
  • The Head element contains language that is used to provide logic and interactivity
  • The body element represents the user of interacting with webpage
